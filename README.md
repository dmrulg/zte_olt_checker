# zte_olt_checker

![image](https://github.com/dmrulg/zte_olt_checker/assets/29884923/7f36956a-a58a-47a6-89b9-2a01af2485e5)

      "10.10.10.10",
      "10.1.2.1",
      "10.0.1.2",
      "10.4.1.2"

  }
}
